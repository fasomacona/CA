Introducción al Módulo

En este módulo avanzaremos desde los modelos clásicos (ARIMA, Suavizado) hacia las técnicas modernas que están revolucionando la analítica predictiva en las empresas. Aprenderás a elegir el modelo correcto según el tipo de negocio, volumen de datos y precisión requerida.

1. Prophet - Librería de Meta (Facebook)

¿Qué es Prophet?

Es una librería de pronóstico automático diseñada para ser fácil de usar por analistas y científicos de datos. Puede manejar tendencias no lineales, estacionalidad múltiple, efectos de holidays y valores faltantes.

Código Completo

from prophet import Prophet
from prophet.plot import plot_plotly

model = Prophet(
    changepoint_prior_scale=0.05,
    yearly_seasonality=True,
    weekly_seasonality=True,
    daily_seasonality=False,
    seasonality_mode='multiplicative'
)
model.fit(df_prophet)
future = model.make_future_dataframe(periods=90, freq='D')
forecast = model.predict(future)
Ventajas
  • Configuración muy sencilla
  • Excelente manejo de estacionalidad
  • Incluye intervalos de confianza
  • Visualizaciones integradas
Desventajas
  • Menor flexibilidad con variables externas
  • Puede sobreajustarse en series muy ruidosas

Casos de uso reales: Walmart (pronóstico de ventas), Uber (demanda de viajes), Airbnb (ocupación), y gobiernos para pronóstico epidemiológico.

2. Machine Learning: XGBoost y Random Forest

Estos modelos convierten el problema de pronóstico en un problema de regresión supervisada mediante técnicas de feature engineering (lags, medias móviles, variables calendáricas, etc.).

Casos Reales

Amazon usa modelos similares para pronosticar demanda de millones de productos. Bancos los usan para predecir transacciones y fraude.

3. LSTM - Deep Learning para Series Temporales

Las redes LSTM son capaces de aprender dependencias a largo plazo, lo que las hace ideales para patrones complejos que otros modelos no capturan.

Caso real: Empresas de energía usan LSTM para pronosticar consumo eléctrico hora a hora.
