📊 Sesión 3: Estadística Descriptiva y Visualización Básica

1. Importar Bibliotecas

import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
import seaborn as sns

# Configuración visual
plt.style.use('seaborn-v0_8')
sns.set_palette("husl")
%matplotlib inline

2. Cargar y Explorar Datos

df = pd.read_csv('ventas.csv', parse_dates=['fecha'])
print(df.shape)
df.head()

3. Estadística Descriptiva

df.describe().round(2)

# Estadísticas por grupo
df.groupby('region')['ventas'].agg(['count', 'mean', 'sum', 'std']).round(2)

🎨 Sesión 4: Visualizaciones con Matplotlib y Seaborn

Visualizaciones Más Importantes

1. Gráfico de Barras

plt.figure(figsize=(10,6))
sns.barplot(data=df, x='region', y='ventas', estimator=np.mean)
plt.title('Ventas Promedio por Región')
plt.show()

2. Gráfico de Líneas (Serie Temporal)

ventas_diarias = df.groupby('fecha')['ventas'].sum()
plt.figure(figsize=(12,6))
plt.plot(ventas_diarias.index, ventas_diarias.values)
plt.title('Evolución de Ventas Diarias')
plt.xlabel('Fecha')
plt.ylabel('Ventas')
plt.grid(True)
plt.show()

3. Histograma y Boxplot

fig, axes = plt.subplots(1, 2, figsize=(14,6))
sns.histplot(df['ventas'], kde=True, ax=axes[0])
sns.boxplot(x='region', y='ventas', data=df, ax=axes[1])
plt.show()

4. Heatmap de Correlaciones

plt.figure(figsize=(10,8))
sns.heatmap(df.corr(numeric_only=True), annot=True, cmap='coolwarm', center=0)
plt.title('Matriz de Correlación')
plt.show()
