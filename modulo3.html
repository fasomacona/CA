🕒 Sesión 5: Conceptos Fundamentales de Series Temporales

Componentes de una Serie Temporal

  • Tendencia (Trend): Movimiento a largo plazo
  • Estacionalidad (Seasonality): Patrones que se repiten
  • Ciclicidad: Fluctuaciones a mayor escala
  • Ruido (Residual): Variación aleatoria

Código: Descomposición de Serie Temporal

from statsmodels.tsa.seasonal import seasonal_decompose
import matplotlib.pyplot as plt

decomposition = seasonal_decompose(df['ventas'], model='additive', period=30)
decomposition.plot()
plt.show()

Prueba de Estacionariedad (ADF Test)

from statsmodels.tsa.stattools import adfuller

result = adfuller(df['ventas'])
print('ADF Statistic:', result[0])
print('p-value:', result[1])

📈 Sesión 6: Modelos de Pronóstico

1. Suavizado Exponencial

from statsmodels.tsa.holtwinters import ExponentialSmoothing

model = ExponentialSmoothing(df['ventas'], trend='add', seasonal='add', seasonal_periods=12)
fit = model.fit()
pronostico = fit.forecast(30)

2. Modelo ARIMA / SARIMA

from statsmodels.tsa.arima.model import ARIMA

model = ARIMA(df['ventas'], order=(1,1,1))
model_fit = model.fit()
pronostico = model_fit.forecast(steps=30)
